# config/ Notes

This folder contains configuration files used by the Python scripts.

- `skill_config.json`: unified config (env + domain/source content + license policy + quality gates + canonicalization). Edit this file.

Typical debugging:

- If strict validation fails on license checks, update `license_policy.json` (or the skill metadata) and re-run:
  - `py langskills_cli.py validate --strict --package`
