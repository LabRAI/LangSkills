# Integrations domain

## 范围（Scope）
- SaaS/API 集成类技能：Slack/Notion/Google 等的官方 API 与常见集成流程
- 目标是“可复现的集成步骤 + 验证方式”，避免私人数据迁移与敏感权限操作

## 来源策略（Source policy）
优先使用：
- 官方 API 文档与开发者指南
- OpenAPI/参考文档（如有公开版本）

## 禁止事项（Out of scope）
- 任何需要登录账号并自动执行的网页操作（见 `SAFETY.md`）
- 需要用户提供敏感凭证并要求保存/上传的流程
