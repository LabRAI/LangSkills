# Docs

这里集中放置：skill 标准、taxonomy、治理机制、FAQ、方向边界等。

- `docs/skill-format.md`：`skill.md` 强制结构与最小要求
- `docs/taxonomy.md`：domain/topic/risk/level 的约定
- `docs/governance.md`：DoD、分层、owner、生命周期
- `docs/lifecycle.md`：stale/archived 的可执行规则与脚本/工作流
- `docs/domains/`：每个 domain 的边界与来源策略
- `docs/repo_inventory.md`：仓库结构与入口清单（entrypoints/产物/命令）
- `docs/curator_llm.md`：curator 整理链路说明 + LLM API 接入点（prompts/fixtures/命令）
- `docs/docker.md`：用 Docker 固定运行环境（可选）
- `docs/meeting_qa.md`：会议问题 Q1–Q6 的可复现回答模板
- `docs/verify_log.md`：每个 Missing/Amb 的验收日志（PASS/FAIL + 命令）
- `docs/tracks/`：面向特定人群/场景的技能路线图（track）
