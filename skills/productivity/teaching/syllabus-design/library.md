# Library

## Syllabus skeleton (copy-paste)

```text
Course title / term / meeting time
Instructor + contact + office hours

Course description

Learning objectives (3–6)
1) ...

Materials
- Textbook / readings / software

Assessments & grading
- Homework: %
- Project: %
- Exam: %
- Rubrics and late policy:

Schedule (week-by-week)
Week 1: ...

Policies
- Academic integrity
- Accessibility / accommodations
- Attendance / participation
- Collaboration policy

Support resources
- Tutoring / writing center
- Mental health / wellbeing (as required)

Version & change log
- v1.0 (YYYY-MM-DD): initial
```

## Quick checklist

```text
- Objectives ↔ assessments aligned
- Grade breakdown + rubric clear
- Deadlines visible + late policy explicit
- Required institutional policy statements included
- Accessible formatting (headings, lists, clear links)
```

## Prompt snippet

```text
You are a teaching assistant. Help draft a syllabus for a university course.

Inputs:
- Course level, audience, and learning objectives
- Assessment plan and schedule constraints
- Required institutional policy statements (if provided)

Output:
- A concise syllabus skeleton + a checklist for ambiguity/compliance risks.
```
