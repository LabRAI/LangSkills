# Examples

## 评分条款示例（可直接改写）

```text
Late policy:
- 24 hours grace per assignment, no questions asked.
- After grace is used, late submissions receive a 10% deduction per day, up to 3 days.
- Extensions require prior approval except documented emergencies.
```

```text
By the end of this course, students will be able to:

```
