# Library

## Lab meeting agenda template (copy-paste)

```text
Lab meeting (YYYY-MM-DD)

Goal(s) for today:
- <sync / deep-dive / decisions>

Agenda (timeboxed)
1) Wins & announcements (5 min)
2) Async updates roundup (5 min)
3) Deep dive (25–35 min)
4) Blockers & decisions (10–15 min)
5) Action items (5 min)

Action items
- [ ] <item> (owner: <name>, due: YYYY-MM-DD, done when: <criteria>)
```

## Retro prompts (monthly)

```text
- What should we stop / start / continue?
- Which part of the meeting produced the most value?
- Are we spending enough time on feedback vs status updates?
```

## Prompt snippet

```text
You are a PI assistant. Help me run a productive lab meeting.

Inputs:
- Team size and meeting duration
- Current milestone + top blockers
- Whether today is sync, paper discussion, or deep dive

Output:
- A timeboxed agenda + roles + an action-item list format.
```
