# Examples

## 单条 comment 的回复格式示例

```text
R2-C3: Reviewer comment (short quote)
We appreciate the reviewer’s suggestion. We revised Section 3.2 to clarify X and added citation Y.

Manuscript changes:
- Section 3.2, paragraph 2 (page 5, lines 12–25)
- Added explanation of X and a new reference to Y.
```

## 语气替换示例

- 不建议：We strongly disagree.
- 更稳妥：We appreciate the concern. Our results suggest ..., and we have clarified ... / added evidence ...
