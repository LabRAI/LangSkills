# Library

## Submission package checklist (copy-paste)

```text
Submission package checklist
- Manuscript PDF (final render checked)
- Source files (LaTeX/Word) + figures
- Supplementary material (if any)
- Cover letter (if required)
- Data/code availability statement + links (if applicable)
- Ethics / IRB / consent statements (if applicable)
- Conflict of interest + funding statements
- Author contributions (if required)
- Suggested reviewers / exclusions (if required)
```

## 1-page outline template

```text
One-page outline
- One-sentence contribution:
- Target audience + venue constraints:
- Key results (3–5 bullets):
- Figure list (F1..Fn: question answered):
- What is new vs prior work:
- Limitations / scope boundary:
```

## Prompt snippet

```text
You are a PI assistant. Help me create a submission checklist and a 1-page outline for a paper.

Inputs:
- Venue type (journal or conference) and any constraints I provide
- My key results and draft figure list
- Coauthor and disclosure constraints (COI, funding, ethics)

Output:
- A concise checklist + a 1-page outline + a final verification checklist.
```
