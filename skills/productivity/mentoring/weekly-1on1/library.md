# Library

## Weekly 1:1 agenda (copy-paste)

```text
Weekly 1:1 (YYYY-MM-DD)

1) Check-in (2–3 min)
2) Progress since last meeting (5–10 min)
3) Blockers / decisions needed (10–20 min)
4) Next steps for the coming week (5–10 min)
5) Action items (3–7 items) + due dates (5 min)

Action items
- [ ] <item> (owner: <name>, due: YYYY-MM-DD, done when: <criteria>)
```

## Running log template

```text
Date | Key decisions | Top blockers | Action items link
```

## Prompt snippet

```text
You are a PI assistant. Help me run a weekly 1:1 with a mentee.

Inputs:
- Current milestone and deadlines
- Mentee's weekly update (wins, blockers, next steps)

Output:
- A focused agenda + a prioritized action-item list (owner, due date, done criteria).
```
