# Library

## Pipeline spreadsheet columns (copy-paste)

```text
Sponsor | Program | Link | Due date | Internal due | Eligibility | Fit | Budget range | Required docs | Status | Next action | Owner | Notes
```

## Weekly triage checklist

```text
- Is this eligible for my career stage and institution?
- Is the fit strong (why this sponsor/program)?
- Is the scope feasible with current resources?
- What is the next action (skip / ask OSP / write 1-pager / recruit collaborator)?
```

## Prompt snippet

```text
You are a PI assistant. Help me set up a funding opportunity pipeline.

Inputs:
- My research keywords and career stage
- A list of candidate funding opportunities (links + deadlines)

Output:
- A pipeline table schema + a weekly triage routine + a 12-month submission calendar sketch.
```
