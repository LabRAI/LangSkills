# Library

## Minimal compliance checklist (copy-paste)

```text
Compliance checklist (draft)
- Program/solicitation matched and eligibility confirmed
- Page limits + font + margins verified
- Required sections present (Project Summary, Project Description, References, etc.)
- Required supplementary documents prepared (as applicable)
- Budget + justification reviewed by OSP
- Facilities/resources/current & pending support handled per instructions
- PDF assembly and upload naming conventions verified
```

## Project Description outline (copy-paste)

```text
1. Motivation and gap
2. Prior work and why now
3. Objectives / hypotheses
4. Research plan (work packages)
   - WP1: ...
   - WP2: ...
5. Evaluation plan and metrics
6. Timeline and milestones
7. Risk management and alternatives
8. Broader impacts and dissemination
9. Team and management plan
```

## Prompt snippet

```text
You are a grant-writing assistant. Help convert an NSF-style solicitation into a compliance checklist and a proposal outline.

Inputs:
- Solicitation link + deadline
- My project summary bullets (IM/BI)
- Team roles + constraints (budget, resources)

Output:
- A checklist + a structured outline (sections + bullets)

Rules:
- Always prioritize compliance (page limits, required documents).
- Every budget item must map to a work package or deliverable.
```
