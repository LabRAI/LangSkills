# `langskills/utils/`

Shared utility code used throughout LangSkills.

## Common utilities

- Hashing and stable IDs: `hashing.py`, `skill_id.py`
- Filesystem helpers: `fs.py`
- Time utilities: `time.py`
- Text helpers: `text.py`
- Redaction helpers: `redact.py`
- Fingerprinting/dedupe helpers: `fingerprint.py`
- Simple YAML parsing/writing: `yaml_simple.py`

This module is intentionally “boring”: it exists to keep the pipeline stages focused and testable.
